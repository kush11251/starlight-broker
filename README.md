# Starlight Broker
A high-performance, real-time event-driven message broker.

## Table of Contents
- [Introduction](#introduction)
- [Getting Started](#getting-started)
- [Run Command](#run-command)

## Introduction
This project is a real-time event-driven message broker written in Go.

## Getting Started
First, navigate to the project directory and initialize the Go module.
```bash
go mod init starlight-broker
```
Then, install the dependencies.
```bash
go get github.com/gorilla/mux
```
## Run Command
To run the project, use the following command:
```bash
go run src/main.go
```
